# 客户端支持平台范围（Windows / Android）

## 摘要

本文档规定本产品 **客户端** 在当前阶段的 **支持平台范围**：**正式支持** **Windows** 与 **Android** 两种终端形态；**iOS**、**Linux**、**macOS**、**Web** 及其它操作系统 **暂不纳入** 开发与交付范围。后续若扩展平台，须通过变更流程修订本文档并同步 **SAD** 与发布说明。

## 关键词

支持平台；Windows；Android；范围外；iOS；Linux

## 1. 范围内（In Scope）

| 平台 | 形态 | 说明 |
|------|------|------|
| **Windows** | 桌面客户端（当前实现为 **WPF**，`.NET` `net9.0-windows`） | 与 Visual Studio 2022 配合开发与发布；数据路径等业务规则见项目记忆。 |
| **Android** | 移动客户端（当前实现为 **.NET Android**，`net9.0-android`） | API 级别与相机扫码等以 `RYIOF_SRS20260424001` 为准。 |

上述两端共享 **同一套 TOTP 核心逻辑**（`TwoFactorAuth.Core`）为当前代码事实；本 SRS **不**要求 iOS/Linux 复用同一技术栈，仅约束 **是否立项交付**。

## 2. 范围外（Out of Scope，现阶段）

以下平台 **不提供** 官方客户端或 **不承诺** 兼容与支持，直至文档另有修订：

- **iOS** / **iPadOS**
- **Linux**（各发行版桌面）
- **macOS**
- **浏览器插件 / 纯 Web 客户端**（未单独立项前一律视为范围外）

## 3. 与其它文档的关系

- TOTP 功能细节（扫码、手动添加、算法与密钥格式）：`RYIOF_SRS20260424001`、`RYIOF_DWS20260424002`。
- 账号、计费、换机：`RYIOF_SRS20260424002`；其实现不得默示要求交付 iOS/Linux 客户端。
